# Ejercicios de Programación ES6 - JavaScript Moderno

## Descripción
Colección de ejercicios prácticos de ECMAScript 6 (ES6) para aprender fundamentos de JavaScript moderno, desde conceptos básicos hasta estructuras de control complejas.

## Estructura de Ejercicios

### Nivel Básico (Inicio ES6)
- **Ejercicio 1** (El2.htm): Suma de dos números variables
- **Ejercicio 2** (Ej3.htm): Incremento de valores numéricos  
- **Ejercicio 3** (Ej4.htm): Uso de constantes y concatenación
- **Ejercicio 4** (Ej5.htm): Multiplicación y corrección de errores
- **Ejercicio 5** (Ej6.htm): Introducción de datos por teclado
- **Ejercicio 6** (Ej7.htm): Conversión de strings a números
- **Ejercicio 7** (Ej8.htm): Suma de números ingresados por usuario

### Nivel Intermedio
- **Ejercicio 8** (Ej9.htm): Manipulación del DOM con innerHTML
- **Ejercicio 9** (Ej10.htm): Bucles while y scope de variables
- **Ejercicio 10** (Condicionales2.htm): Comparación de números
- **Ejercicio 11** (Condicionales3.htm): Cálculo de notas y promedios
- **Ejercicio 12** (Descuentos.htm): Sistema de descuentos condicionales

## Tecnologías
- HTML5
- ECMAScript 6 (ES6)
- DOM Manipulation
- Browser APIs

## Conceptos Cubiertos
- Variables (let, const, var)
- Estructuras de control (if-else, while)
- Manipulación del DOM
- Interacción con usuario (prompt, alert)
- Conversión de tipos de datos
- Depuración de errores comunes

## Cómo Ejecutar
1. Clona o descarga el repositorio
2. Abre el archivo HTML correspondiente al ejercicio
3. Ejecuta en navegador moderno (Chrome, Firefox, Edge)

## Errores Corregidos
- window.pr0mpt → window.prompt
- $(variable) → ${variable} o concatenación
- = en comparaciones → == o ===
- Variables no declaradas → Scope correcto

## Progresión Recomendada
1. Ejercicios 1-3: Conceptos básicos
2. Ejercicios 4-7: Entrada de datos  
3. Ejercicios 8-12: DOM y condicionales
